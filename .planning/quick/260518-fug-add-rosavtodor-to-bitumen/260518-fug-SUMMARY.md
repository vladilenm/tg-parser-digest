# Quick Task 260518-fug — Summary

## Что сделано

В `CLASSIFY_SYSTEM_PROMPT` (Pass 1) расширен список ключевых слов категории `bitumen`: добавлено «Росавтодор» в конец перечисления.

Было:
```
bitumen — битум и дорожно-строительные нефтепродукты: битум, ПБВ, асфальт, битумные вяжущие, bitumen
```

Стало:
```
bitumen — битум и дорожно-строительные нефтепродукты: битум, ПБВ, асфальт, битумные вяжущие, bitumen, Росавтодор
```

## Почему так

- Заказчик прислал новость про Росавтодор vs. цементные дороги и попросил, чтобы такие новости попадали в `#битум`.
- Росавтодор — федеральный регулятор автодорожной отрасли; любая его повестка релевантна дорожному строительству → битум/асфальт.
- Pass 2 (`buildSummarizeCategoryPrompt`) **не правился** — он не перечисляет per-category ключевые слова, только per-mentions компании. На Pass 2 пост уже отнесён к категории, нужен только дословный экстракт.
